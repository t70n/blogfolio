Bomber7 is a local multiplayer game inspired by the classic Bomberman, with a twist: the maps and themes are based on ENSEEIHT, our engineering school. Built from scratch by a team of six, we created a fun, competitive arcade-style experience for 1–4 players.

## Project Overview

![Bomber7 Bonus](/assets/legacy/img/bomber7_bonus.png)

**Goal**: Create an engaging local multiplayer game with customizable controls, dynamic power-ups, and AI opponents.

## Core Gameplay

- **Classic Bomberman mechanics**: Place bombs, destroy enemies and blocks, survive to win
- **Four custom maps** inspired by ENSEEIHT locations (courtyard, student lounge, etc.)
- **Power-ups**: Extra bombs, speed boosts, manual detonation, and more
- **Local multiplayer**: Play with friends on the same keyboard or multiple keyboards
- **AI opponents**: Simple bot logic for solo or team play

## Technical Stack

- **LibGDX**: Java-based game engine for rendering, input handling, and physics
- **MVC Architecture**: Clean separation of game logic (Model), UI (View), and input handling (Controller)
- **Design Patterns**: Singleton (resource management), Factory (map generation), Observer (event handling), Strategy (AI behavior)
- **Agile Workflow**: GitHub Projects for task tracking, CI/CD pipelines for testing, Scrum sprints

## User Experience

![Gameplay Preview](/assets/legacy/img/bomber7_home.png)
